## Streamlit

- **说明**：Streamlit是一个开源的python库，专为数据工程师及机器学习工程师设计，用来快速基于python代码构建交互式的web网站，且无需会前端技术

- **使用步骤**：
  1. 安装 `pip install streamlit`
  2. 引入 `import streamlit as st`
  3. 调用streamlit api来构建网页
  4. 运行：`streamlit run xxx.py`

- API 查询在官方文档的<a href="https://docs.streamlit.io/develop/api-reference">API Reference</a>处 

> Streamlit只适用于数据分析，人工智能学习可视化等领域制作网页
> 真正的商业业务网页依然需要正式的前端技术
