**本笔记主要记录与Java不同的语法**


## 字面量与变量

- Python是动态类型语言, 一个变量可以根据接收的值, 自动转换类型

- boolean: `True` , `False`  -- 开头必须大写

- 空值: `None` -- 不是null, 且开头要大写

- 数字, 字符串省略

- 打印变量: `print(变量)`
- 查看类型: `type(变量)` -- 查看变量类型
- 判断类型: `isinstance(变量, 类型)` -- 如果是类型, 则返回 True

> boolean类型的值本质是整数, 所以可以做加减运算
>       print(True + 1) ==> 2
>       print(False + 10) ==> 10


### 字符串

- 双引号`""`
```py
    s1 = "Hello"
    s1 = "I'm Allen"    # 双引号方便包含单引号的内容, 省去转译
```
- 单引号`''`
```py
    s2 = 'Python'
    s2 = 'He said "hello"'  # 单引号方便包含双引号的内容, 省去转译
```
- 三引号`"""`
```py
s3 = """
    Hello World
    Hello Java
    """             # 三引号方便对文本进行排版
```

- 字符串拼接: 

  - 大括号拼接`f + {变量}`  ⭐⭐⭐
    ```py
    f"My name is {name}, I'm {age} years old."  # 要注意, 此方式必须 f 开头
    ```

  - 字符串之间用`+`拼接, 非字符串类型需要用`str()`转换成字符串
    ```py
    "My name is " + name + ", I'm " + str(age) + " years old."
    ```

  - 占位符拼接`%s`
    ```py
    "My name is %s, I'm %s years old." % (name, age)
    ```
