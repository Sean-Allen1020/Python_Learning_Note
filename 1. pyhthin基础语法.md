**本笔记主要记录与Java不同的语法**

## 变量

### 字面量与变量

- Python是动态类型语言, 一个变量可以根据接收的值, 自动转换类型

- boolean: `True` , `False`  -- 开头必须大写

- 空值: `None` -- 不是null, 且开头要大写

- 数字, 字符串省略

- 打印变量: `print(变量)`
- 查看类型: `type(变量)` -- 查看变量类型
- 判断类型: `isinstance(变量, 类型)` -- 如果是类型, 则返回 True

> boolean类型的值本质是整数, 所以可以做加减运算
>       print(True + 1) ==> 2
>       print(False + 10) ==> 10

---------------------------

### 字符串

- 双引号`""`
```py
    s1 = "Hello"
    s1 = "I'm Allen"    # 双引号方便包含单引号的内容, 省去转译
```
- 单引号`''`
```py
    s2 = 'Python'
    s2 = 'He said "hello"'  # 单引号方便包含双引号的内容, 省去转译
```
- 三引号`"""`
```py
s3 = """
    Hello World
    Hello Java
    """             # 三引号方便对文本进行排版
```

## 运算逻辑与拼接

### 字符串拼接: 

  - 大括号拼接`f + {变量}`  ⭐⭐⭐
    ```py
    f"My name is {name}, I'm {age} years old."  # 要注意, 此方式必须 f 开头
    ```

  - 字符串之间用`+`拼接, 非字符串类型需要用`str()`转换成字符串
    ```py
    "My name is " + name + ", I'm " + str(age) + " years old."
    ```

  - 占位符拼接`%s`
    ```py
    "My name is %s, I'm %s years old." % (name, age)
    ```
> 类型转换函数：类型()
>   int()
>   float()
>   boolean()

### 算术运算符 与 逻辑运算符

- `//`：整除 --> 因为python没有类型，所以整除是必要的
- `**`：幂指数 --> `10**3` 10的3次方
- `==`：含义没变，只是python中，**字符串**的比较也是用 ==
- python只有`+=1`, `-=1`，没有`++`, `--`

> 0.5次方就是开根，可以利用这点直接计算开根

- `and` --> `&&`
- `or`  --> `||`
- `not` --> `!`

> python将传统的逻辑符号改成了英语

## 流程控制语句

### if

- python中的 if 是**根据缩进**来判断代码块的
- python中的 if 没有 () 与 {}

```py
  if calendar.isleap(year):
      print(f"{year}是闰年")
  else:
      print(f"{year}是平年")
```

> calendar.isleap() --> python的内置方法

---------------------------

### match ... case
- 类似java的 switch...case
- case中 | 表示 or
- 没有& 表示 and 的语法
- case行内可以**嵌套if语句**

```py
day = input("请输入星期几")

match day:
    case 1:
        print("周一：周例会日")
    case 2:
        print("周二：学习培训日")
    case 3:
        print("周三：项目开发日")
    case 6 | 7:             # | 为 or，注意：没有 & and 的语法
        print("休息日")
    case _:                 # _ 类似 java的 default
        print("输入错误")

#--------------------------------------------#
x = int(input("请输入数字"))

match x:
    case 1 | 2:
        print("x 是 1 或 2")
    case n if n > 10:     # case行内可以嵌套 if语句
        print("x 大于 10")
    case _:
        print("其他情况")
```

---------------------------

### while

- else 的部分，可写可不写
- break 和 continue 的用法与Java一样
- 
```py
while 条件表达式:
  执行语句

else:
  循环正常结束时的执行语句
```
  
- 示例代码
```py
sum = 0
i = 1
while i <= 100:
    if i % 2 == 0:
        sum += i
    i += 1

print(sum)

```
