## 入门程序

### 1. 查看机器人协议：

`https://www.tiobe.com/robots.txt`
User-agent: *
Disallow: /wp-admin/
Allow: /wp-admin/admin-ajax.php
Sitemap: https://www.tiobe.com/sitemap_index.xml

----------------------------------

### 2. 安装requests库

- 项目根目录执行 `pip install requests`
- 导入 `import requests`

**说明**：
`requests` 是 python 很常用的发送`HTTP 请求`的库。
让 Python 去访问网页、接口、下载内容的工具。

- 爬虫里常用它来做这些事：

  - 向某个网址发请求
  - 获取网页源码
  - 获取接口返回的 JSON 数据
  - 带请求头、参数、cookie 去访问页面

----------------------------------

### 3. 获取页面整体 -- html原始页面

```py
  import requests

  # 定义url
  target_url = "https://www.tiobe.com/tiobe-index/"

  # 发送请求，获取响应数据对象
  response = requests.get(target_url)

  # 输出数据的文本到控制台
  print(response.text)  # 输出页面的原始html文本内容

```
