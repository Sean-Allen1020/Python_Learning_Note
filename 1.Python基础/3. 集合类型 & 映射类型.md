### 集合 set

**特点**：
- 无序，不可重复 --> **自动去重**
- 可修改

**语法**：`集合名 = {元素1, 元素2, 元素3, ...}` ==> `s = {15, 20, 32, 13, True, "你好"}`
          `s = set()` ==> 空集合

**方法**：
- `.add(元素)`                  添加元素到集合中
- `.remove(元素)`               移除集合中的指定元素，元素不存在会报错
- `.pop()`                      随机删除集合中的一个元素并返回
- `.clear()`                    清空集合

- `s1.difference(s2)`           求两个集合的差集（包含在 `s1` 中但不包含在 `s2` 中的元素）
- `s1.union(s2)`                求两个集合的并集
- `s1.intersection(s2)`         求两个集合的交集
- `s1.isdisjoint(s2)`           判断两个集合是否没有交集，没有则返回 True

- `s1 - s2`                     差集,包含在 `s1` 中但不包含在 `s2` 中的元素）
- `s1 | s2`                     并集
- `s1 & s2`                     交集
- `s1 ^ s2`                     对称差集，两个集合差集的汇总

- 集合转列表：`list1 = [*set1]` 利用解包，可以将集合转换成列表

> 补充：集合推导式
> `{想要添加的元素 for s in set1 if 条件}`  --> 在集合内进行 for循环，if判断等操作来获取元素，直接推导出一个新集合

### 字典 dict

**特点**：
- 键值对存储，键不可重复，可修改
- 字典中可以嵌套字典，实际上整个结构就类似一个 json数据

**语法**：`字典名 = {key1: value1, key2: value2, ...}` ==> `d = {"可乐": 3, "牛奶": 5}`
          `d = dict() , d = {}` ==> 空字典

**操作**：

- `val = 字典名[key]`   -- 根据键获取值
- `字典名[key] = xxx`   -- 修改值

| 分类 | 写法                      | 作用                                                                     |
| ---- | ------------------------- | ------------------------------------------------------------------------ |
| 添加 | `字典名[key] = value`     | 当 key 不存在，则添加键值                                                |
| 修改 | `字典名[key] = value`     | 当 key 存在，则修改值                                                    |
|      |                           |                                                                          |
| 删除 | `字典名.pop(key)`         | 删除字典中指定的 key，并返回该 key 对应的 value                          |
| 删除 | `del 字典名[key]`         | 删除字典中指定的键值对                                                   |
|      |                           |                                                                          |
| 查询 | `字典名[key]`             | 根据 key 获取 value                                                      |
| 查询 | `字典名.get(key, 默认值)` | 根据 key 获取 value。**如果key不存在，则添加key并赋默认值**              |
| 查询 | `字典名.keys()`           | 获取所有的 key                                                           |
| 查询 | `字典名.values()`         | 获取所有的 value                                                         |
| 查询 | `字典名.items()`          | 获取所有的 key-value 键值对，返回一个元组，0索引为键 1索引为值，以此类推 |

> 字典本质类似一个json数据，所以可以字典存字典来套娃
>   `字典名[key1] = {key2: val2, key3: val3}`
