## 函数

### 函数基础

**特点**：

- `def` 关键字用于定义函数
- 函数执行时，必须**先定义后调用** (单纯函数定义的时候，调用顺序无所谓)
- **形参甚至可以是函数**
- **返回值可以有多个**

```py
    def function(param, func_)  # 函数可以直接作为形参被传入
        # function body

        返回值1, 返回值2= func_(param)
        return 返回值1, 返回值2, ...   # 可有可无

    
    result = function(para)  # 调用
```
> 多个返回值会被封装到 元组 中
> 可用一个变量接取，也可以通过解包分别接取

**函数说明文档**
- 在函数内，用 `"""`包裹形式，可以书写函数说明文档
- 在调用函数时，把鼠标移到函数处，即可展示说明文档

```py
    def function(param)
        """
        函数内容描述....

        :param 形参: 形参解释
        :return 返回值: 返回值解释
        """
        # ....处理
        # return ...
```

### 函数进阶

**变量作用域**：
- 使用`global`关键字，可以让函数内使用全局变量


**关键字传参**
- 位置传参：`func_(a, b, c)` ==> 传统传参，参数顺序必须与函数一致
- 关键字传参：  `func_(b=200, a=100, c=300)` ==> 直接在传参时赋值，参数顺序随意，但参数名必须与函数一致
- 位置参数必须在关键字参数前面
- 形参里单独的`*`代表之后的参数必须是关键字参数：`func_(a, b, *, c, d)` ==> c, d 必须是关键字参数


**默认参数**
- 默认参数设置：`def func_(a, b, c=300)` ==> 在定义函数时设置默认参数值，如果调用时没有传入对应参数，则使用默认值

> 默认参数的后面不能跟常规参数，否则会报错


**不定长参数**
- 位置参数的不定长：`def func_(*args)` ==> 形参前面加一个 *。参数被传入封装成元组
- 关键字参数的不定长：`def func_(**kwargs)` ==> 形参前加两个 *。参数被传入封装成字典

- 不定长参数代码示例：
```py
    # 最好用 .get() 来获取值，直接用 字典[key] 的话，key为None时会直接报错
    def calc_data(*args, **kwargs):
        min_data = min(args)
        max_data = max(args)
        avg_data = sum(args) / len(args)

        float = kwargs.get("round")
        if float is None:
            float = 1

        if kwargs.get('print'):
            print(f"最小值：{min_data}，最大值：{max_data}，平均值：{round(avg_data, float)}")
            return None
        else:
            return min_data, max_data, round(avg_data, float)


    calc_data(12,27,34, print=True, round=2)
    data = calc_data(12, 27, 34, round=3)
    print(data)
    data = calc_data(12, 27, 34)
    print(data)
```

### 匿名函数

**说明**：逻辑简单的函数(单行代码)，可以使用匿名函数定义

**语法**：`lambda 形参: 函数体` --> 没有形参的时候，用空格

**调用**：`变量名 = lambda 形参: 函数体`
          `变量名(传参)`

- 示例
```py
    output_line = lambda : print("--------------")
    add = lambda x, y: x + y

    output_line()
    print(add(20, 30))
```

- 实际运用
```py
    words = ["apple", "sky", "elephant", "go", "book", "a", "dictionary", "blue"]
    words.sort(key= lambda word: len(word))    # lambda匿名函数写法
    words.sort(key= len)                       # 将函数作为参数传入的写法
    print(words)
```

### 类型注解

**语法**：`变量: 类型`

**函数的类型注解**：`def func_(参数: 类型) -> 返回值类型:`

```py
    # 基本类型
    a: int = 695
    score: float = 98.5
    hobby: str = "Python"
    flag: bool = True
    pic: None = None

    # 数据容器类型
    names: list[str | int] = ["A", "C", "E", 100]
    phones: set[str] = {"13309091111", "15209109121"}
    options: dict[str, int] = {"count": 0, "total": 0}
    goods: tuple[str, int, int] = ("手机", 5999, 1)

    # 特殊类型
    data: Any = None        # 表示可以接收任意类型的参数
    key: Callable[[参数类型], 返回值类型]   # 表示只能接收函数类型的参数，[]中表示函数接形参类型和返回值类型
```
