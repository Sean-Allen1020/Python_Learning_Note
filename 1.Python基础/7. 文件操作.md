### 读写文件

```py
    # 1. 打开文件
    f = open("文件", "r 或者 w", encoding="utf-8") # r 是读取模式，w 是写入模式

    # 2. 读取/写入文件
    text = f.read()         # 读所有内容
    text = f.readlines()    # 一行一行的读（需要用 .strip() 来去除空格）
    print(text)

    f.write("写入文本") # 写

    # 3. 释放资源
    f.close()
```
> 为了保证即便出错的时候，资源也可以成功释放，可以用`try-except-finally`来解决 -- 其中，`except`可以省略
> 
> **除了`try-finally`外，还可以用`with`来解决资源释放问题**

### with -- 用于文件读写的资源释放

`with` 是用于管理 “进入某个对象并在用完后自动收尾” 的过程。
   - 打开一个临时工作区域 --> 在里面操作 --> 结束后自动释放资源

- **使用场景**：文件读写 --> 文件操作后需要关闭，`with`可以自动关闭
- **语法**：`with 对象实例 as 别名:`
- **示例**：

```py
    with open("a.txt", "r", encoding="utf-8") as f:
        data = f.read()
        print(data)
```

### Json格式文件读写 ⭐⭐⭐

- json序列化：  `json.dump()`
- json反序列化：`json.load()`

```py
    obj = {
        'name': 'Jack',
        'age': 18,
        'gender': '男'
    }

    # 序列化
    with open("xxx.json", "w", encoding="utf-8") as f:
        json.dump(obj, f, ensure_ascii=False, indent=4)
        
        # f 是文件对象
        # ensure_ascii=False 是防止汉字被Unicode转义
        # indent=2 是每级缩进4空格

    # 反序列化
    with open("xxx.json", "r", encoding="utf-8") as f:
        user = json.load(f)
        print(user)
