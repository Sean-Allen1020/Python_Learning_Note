### 读写文件

```py
    # 0. 判断文件夹存在与否，不存在则创建文件夹
    os.makedirs("文件夹路径", exist_ok=True)  # 多级路径创建，来保证父级路径的存在，如果路径已经存在则跳过该代码

    # 1. 打开文件
    f = open("文件路径", "r 或者 w 或者 a", encoding="utf-8") # r 是读取模式，w 是写入模式，a 是追加模式

    # 2. 读取/写入/追加
    text = f.read()                     # 读所有内容
    text = f.readlines()                # 一行一行的读（需要用 .strip() 来去除空格）
    for line in f: print(line.strip())  # 直接通过遍历文件对象，来一行一行的读

    f.write("写入文本") # 写入/追加 --> 区别只是重置文件内容和续写文件内容

    # 3. 释放资源
    f.close()
```

> 为了保证即便出错的时候，资源也可以成功释放，可以用`try-except-finally`来解决 -- 其中，`except`可以省略
> 除了`try-finally`外，还可以用`with`来解决资源释放问题

### with -- 用于文件读写的资源释放

`with` 是用于管理 “进入某个对象并在用完后自动收尾” 的过程。
   - 打开一个临时工作区域 --> 在里面操作 --> 结束后自动释放资源

- **使用场景**：文件读写 --> 文件操作后需要关闭，`with`可以自动关闭
- **语法**：`with 对象实例 as 别名:`
- **示例**：

```py
    with open("a.txt", "r", encoding="utf-8") as f:
        data = f.read()
        print(data)
```

### 文件路径操作库 -- `pathlib` ⭐⭐⭐

传统的python路径是用`os.path`来进行操作的，`pathlib`提供了一个更动态的路径操作库，方便路径相关代码书写

- `Path(__file__).resolve()` -- 用于获取代码所在文件的绝对路径，加上`.parent`便能获取父级目录
- 基于此，便可以对路径进行拼接或定位

**示例**：
- 假如文件路径为：`C:\AI-Partner\program\chat_service.py`

```py
    from pathlib import Path

    BASE_DIR = Path(__file__).resolve().parent      # program/  -- 获取代码所在文件的父级路径，以此为基底
    PROJECT_DIR = BASE_DIR.parent                   # AI-Partner/ -- 父级的父级
    RESOURCE_DIR = PROJECT_DIR / "resources"        # AI-Partner/resources/
    SESSION_DIR = BASE_DIR / "session_data"         # program/session_data/

    # 判断文件夹存在与否，不存在则创建文件夹。
    #父级路径的处理与 os.makedirs("文件夹路径", exist_ok=True) 相同
    Path("文件路径").mkdir(parents=True, exist_ok=True)  
```

### Json格式文件读写 ⭐⭐⭐

- json序列化：  `json.dump()`
- json反序列化：`json.load()`

```py
    obj = {
        'name': 'Jack',
        'age': 18,
        'gender': '男'
    }

    # 序列化
    with open("xxx.json", "w", encoding="utf-8") as f:
        json.dump(obj, f, ensure_ascii=False, indent=4)
        
        # f 是文件对象
        # ensure_ascii=False 是防止汉字被Unicode转义
        # indent=2 是每级缩进4空格

    # 反序列化
    with open("xxx.json", "r", encoding="utf-8") as f:
        user = json.load(f)
        print(user)
