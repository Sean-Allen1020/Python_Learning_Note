## 数据存储容器

顾名思义，是可以存储多个数据到容器

依据：是否支持重复，是否可以修改，是否有序，分为：

- **序列类型**：有序，可切片
  - `list`
  - `str`
  - `tuple` (元组)
  
- **集合类型**：无序，不可重复
  - `set`

- **映射类型**：键值对存储数据，通过键访问值
  - `dict` (字典)


### 列表 list

**语法**：`列表名 = [元素1, 元素2, 元素3, ...]` ==> `s = [15, 20, 32, 13, True, "你好"]`

**特点**：
- 可存储不同类型的元素
- 元素有序，可重复，可修改
- python有反向索引的功能：
    - 正常索引： 0，1，2，3...
    - 反向索引：-1，-2，-3...   可从列表的末尾往前，进行反向操作

**操作**：
- 删除：`del s[idx]`

- 切片：`列表名[startIdx:endIdx:step]`，有序类型特有的操作，可截取部分元素 ⭐⭐
    
      - 包头不包尾
      - step为间隔数
      - `startIdx`为 0 或 -1，可以省略
      - `step`为 1，可以省略

- 解包：`*列表名` --> 将列表解单个元素 ⭐⭐
- 组包：`new_list[*list1, *list2]`--> 将不同解包的列表拼接成一个新列表

- 拼接：`列表1 + 列表2`

> 判断元素是否存在列表中：`if item in item_list`  ⭐⭐
> 
> 列表推导式：
> `[想要添加的元素 for i in items if 条件]`  ⭐⭐
> 示例：
> `new_list = [i*2 for i in num_list if i % 2 == 0]`  --> 在列表内进行 for循环，if判断等操作来获取元素，直接推导出一个新列表 

**方法**：
- `.append(元素)`       尾部追加元素
- `.insert(idx, 元素)`  指定索引之前，插入元素
- `.remove(值)`         移除列表中，第一个匹配到的值
- `.pop(idx)`           移除列表中的指定索引元素，并返回该元素（不写，则默认最后一个）
- `.sort()`             列表元素排序，元素类型必须一致
- `.reverse()`          反转列表元素，类似 Java集合反转`Collections.reverse(list)`

- `min(列表名)`         最小值
- `max(列表名)`         最大值

- `len(列表名)`         长度
- `sum(列表名)`         求和

### 字符串 str

**特点**：
- 元素有序，可重复，可切片
- 不可修改

**方法**：
- `.find(子串)`               在字符串中查找子串，返回**第一次**出现的索引位置，找不到返回 `-1`，类似 Java的`.indexOf("")`
- `.count(子串)`              统计子串在字符串中出现的次数
- `.upper()`                  将字符串中的所有字母转换为大写
- `.lower()`                  将字符串中的所有字母转换为小写
- `.split(分隔符)`            将字符串按指定分隔符分割成列表，与 Java的`.split()`相同
- `.strip()`                  去除字符串两端的空白字符或指定字符
- `.replace(旧子串, 新子串)`   将字符串中的指定子串替换为新的子串
- `.startswith(子串)`         检查字符串是否以指定子串开头，返回布尔值
- `.endswith(子串)`           检查字符串是否以指定子串结尾，返回布尔值
  
- 字符串反转：`str_reverse = str[::-1]`  Java中的类似方法为：`new StringBuilder(str).reverse().toString()`

**示例**：
```py
    # 回文字符串
    str = "上海自来水来自海上"
    str_reverse = str[::-1]     # 切片 + 反向索引，获取反转字符串
    print(str_reverse == str)


    # 邮箱验证
    mail = "zhuzhuzhu11330@126.com"
    if mail.count("@") == 1 and "." in mail:    # 判断元素是否存在列表中：`if item in item_list`
        print(mail)
        print("right email")
    else:
        print("wrong email")
```

### tuple 元组

**语法**：`元组 = (元素1, 元素2, 元素3, ...)` ==> `t = (15, 20, 32, 13, True, "你好")`

**特点**：
- 元素有序，可重复，可切片
- **不可修改** --> 与 列表 最大的区别

**方法**
- `.count(元素)`      统计元素在元组中出现的次数
- `.index(元素)`      查找元素在元组中第一次出现的索引

> 定义单元素元组时需要加 , 
>   `t = (100, )`
> 否则会报错，因为单纯括号的话，会被识别为 int整数

**解包 组包**

- 组包： `t = (5, 7, 9, 1)`

- 基础解包：`a, b, c, d = t`  ==> abcd 依次获得对应位置的元素

- 扩展解包：`x, *y, z = t`  ==> x, z获取对应位置的元素，*y 以**列表形式**获取剩余的元素
> '*' 代表收集剩余所有元素
>
> 解包式可以直接写在 for循环行内 `for x, *y, z in t`

```py
  # 利用元组解包组包的元素交换
  a, b, c = 100, 200, 300
  c, a, b = a, b, c

  print(f"{a}, {b}, {c}")
```

### 列表 元组 综合练习

```py
  # 学生成绩元组
  students = (
      ("S001", "张三", 85, 92, 78),
      ("S002", "李四", 92, 88, 95),
      ("S003", "王五", 78, 85, 82),
      ("S004", "宋六", 88, 79, 91)
  )

  # 1. 计算学生平均分
  for id, name, *score in students:   # 解包可以直接用在 for循环行内
      avg = sum(score)/len(score)
      print(f"{name}: {avg:.1f}分")

  # 2. 分别获取 语文 数学 英语 成绩
  chinese = [s[2] for s in students]  # 通过列表推导式来获取
  math    = [s[3] for s in students]
  english = [s[4] for s in students]
  print(f"语文：{chinese}")
  print(f"数学：{math}")
  print(f"英语：{english}")
```
