## 正则表达式

**导入`import re`**

- `re.match(r"表达式", str)` --> 从字符串开头开始匹配，返回 `Match对象`         -- 类似 `^...`
- `re.fullmatch(r"表达式", str)` --> 从字符串开头到结尾完全匹配，返回`Match对象`-- 类似 `^...$` 
- `re.search(r"表达式", str)` --> 从任意位置开始匹配，匹配到第一个匹配项后直接返回 `Match对象`
- `re.findall(r"表达式", str)` --> 从任意位置开始匹配，匹配到所有匹配项后，返回`list`

- `.group(组号)` --> 用于获取`Match对象`匹配到的内容 ⭐⭐       -- `re.findall(r"(\d{4})-(\d{2})-(\d{2})", s).group(2)`，输出为 (\d{2}) 里的内容
- `.span()` --> 用于获取`Match对象`匹配到的内容的 起始索引，结束索引
- `.start()` --> 用于获取`Match对象`匹配到的内容的 起始索引
- `.end()` --> 用于获取`Match对象`匹配到的内容的 结束索引


### 表达式

| 表达式     | 描述                                                    |
| ---------- | ------------------------------------------------------- |
| 普通字符   | 字母、数字、汉字及大多数符号，直接匹配自身              |
| `.`        | 匹配任意一个字符（除 `\n`）                             |
| `\d`       | 匹配数字 `0-9`                                          |
| `\D`       | 匹配非数字                                              |
| `\w`       | 匹配单词字符，即 `a-z`、`A-Z`、`0-9`、`_`、其他语言字符 |
| `\W`       | 匹配非单词字符                                          |
| `[aeiou]`  | 匹配其中的任何单个字符                                  |
| `[^aeiou]` | 求反，匹配不在字符列表中的任何单个字符                  |
| `[0-5]`    | 表示范围                                                |

> `\w` 不想匹配其他语言时，可以添加一个参数：`re.findall(r"\w+", s1, re.ASCII)` 



| 表达式  | 描述                                      |
| ------- | ----------------------------------------- |
| `*`     | 出现任意次（0次或无数次）                 |
| `+`     | 至少出现1次（1次或无数次）                |
| `?`     | 至多出现一次（0次或1次）                  |
| `{m}`   | 出现m次                                   |
| `{m,}`  | 至少出现m次                               |
| `{m,n}` | 出现m到n次                                |
| `\|`    | 或，匹配左右任意一个表达式                |
| `()`    | 分组，将括号里的多个字符视为一个单元      |
| `^`     | 匹配整个字符串的开头，类似`re.match(...)` |
| `$`     | 匹配整个字符串的结尾                      |


```py
s1 = "电话号码：18809090000"
print(re.findall(r"^1[3-9]\d{9}", s2))    # ^ 从字符串的起点开始匹配 -- 开头非1，结果为[]
print(re.findall(r"1[3-9]\d{9}$", s2))    # $ 匹配字符串的结尾 -- 结尾为18xxx，结果为[18809090000]
print(re.findall(r"^1[3-9]\d{9}$", s2))   # 匹配字符串的从头到尾，开头非1，结果为[]


s2 = "邮箱为python666@163.com。"
print(re.findall(r"\w+@\w+\.\w+", s1))              # 结果：[邮箱为python666@163.com]
print(re.findall(r"\w+@\w+\.\w+", s1, re.ASCII))    # 不匹配非ASCII字符(中文汉字等)：[python666@163.com]


s3 = "现在的时间是2026-02-06 10:05:25，今天的天气还可以，气温是28度 "
print(re.findall(r"\d{4}-\d{2}-\d{2}", s3))           # 匹配日期['2026-02-06']
print(re.findall(r"(\d{4})-(\d{2})-(\d{2})", s3))     # 只匹配分组内容，结果为[('2026', '02', '06')]
```
