## 函数

### 函数基础

**特点**：

- `def` 关键字用于定义函数
- 函数执行时，必须**先定义后调用** (单纯函数定义的时候，调用顺序无所谓)
- 返回值可以有多个

```py
    def function(param)
        # function body

        return 返回值1, 返回值2, ...   # 可有可无

    
    result = function(para)  # 调用
```
> 多个返回值会被封装到 元组 中
> 可用一个变量接取，也可以通过解包分别接取

**函数说明文档**
- 在函数内，用 `"""`包裹形式，可以书写函数说明文档
- 在调用函数时，把鼠标移到函数处，即可展示说明文档

```py
    def function(param)
        """
        函数内容描述....

        :param 形参: 形参解释
        :return 返回值: 返回值解释
        """
        # ....处理
        # return ...
```

### 函数进阶

**变量作用域**：
- 使用`global`关键字，可以让函数内使用全局变量

**关键字传参**
- 传统位置传参：`func_(a, b, c)` ==> 传统传参，参数顺序必须与函数一致
- 关键字传参：  `func_(a=100, b=200, c=300)` ==> 直接在传参时赋值，参数顺序随意，但参数名必须与函数一致

**默认参数**
- 默认参数设置：`def func_(a, b, c=300)` ==> 在定义函数时设置默认参数值，如果调用时没有传入对应参数，则使用默认值

> 默认参数的后面不能跟常规参数，否则会报错
