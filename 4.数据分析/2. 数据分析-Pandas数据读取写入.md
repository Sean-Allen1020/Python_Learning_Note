## 数据读取与写入

**读取数据源，创建 DataFrame**：
`df = pd.read_xxx("路径", usecols=[所需列名])` --> xxx 为数据源类型，如：csv、sql、excel、html...

**将DataFrame生成为指定数据类型**
`df.to_xxx("路径", index=False)` --> xxx为想生成的数据类型，将DataFrame转换为实际的文件或文本或数据库内容

**示例**
```py
    import pandas as pd

    # 读取csv文件                               # 获取数据源中所需的列
    df = pd.read_csv("data/sales.csv", usecols=["订单号", "产品类别", "产品名称", "销售数量", "单价"])

    # 数据处理 -- 每个商品的销售额
    df["销售额"] = df["销售数量"] * df["单价"]  # 添加新列

    # 写入数据
    df.to_csv("data/sales02.csv", index=False)  # index=False 不写入索引
```
