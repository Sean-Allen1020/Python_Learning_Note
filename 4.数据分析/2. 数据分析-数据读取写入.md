## 数据处理

### 读取与写入

**读取数据源，创建 DataFrame**：
`df = pd.read_xxx("路径", usecols=[所需列名])` --> xxx 为数据源类型，如：csv、sql、excel、html...

**将DataFrame生成为指定数据类型**
`df.to_xxx("路径", index=False)` --> xxx为想生成的数据类型，将DataFrame转换为实际的文件或文本或数据库内容

**示例**
```py
    import pandas as pd

    # 读取csv文件                               # 获取数据源中所需的列                                # 读取10行
    df = pd.read_csv("data/sales.csv", usecols=["订单号", "产品类别", "产品名称", "销售数量", "单价"], nrows=10)

    # 数据处理 -- 每个商品的销售额
    df["销售额"] = df["销售数量"] * df["单价"]  # 添加新列

    # 写入数据
    df.to_csv("data/sales02.csv", index=False)  # index=False 不写入索引
```

### 查看与选择，以及过滤

**查看**的常见属性与方法：
```py
# 查看
df1.info()
df1.head(n) # 查看前n行数据，默认5行
df1.tail(n) # 查看结尾n行数据
df1.describe() # 查看数值列的统计描述，如数据条数，最小值，最大值，平均值，中位数，标准差等
df1.columns # 获取表头
df1.values  # 获取所有值
df1.shape   # 获取行数和列数

# 选择
df1[列名]   # 操作单列
df1[[列名1, 列名2, ...]]    # 操作多列
df1.iloc[start: stop: step] # 基于行号完成切片      -- 行号不在表上，不可修改，永远是 01234... 
df1.loc[start: stop: step] # 基于索引完成切片(包尾)  -- 索引是表最左端的列，可以自定义，可修改

# 过滤 -- df[条件]
df1[df1['列名'] >= 10]
df1[df1['列名'].isin(['列名1', '列名2'])]
df1[df1['列名'].between(50, 200)]

&   # 和
|   # 或
```

### 清洗

- 清洗的数据一般为：缺失数据，异常数据，重复数据

```py
import pandas as pd
# 配置默认显示行数
pd.set_option('display.min_rows', 30)
df = pd.read_csv("data/sales.csv")

# 缺失值处理
df.isnull() # 查看缺失值
# 删除缺失值
df_drop = df.dropna() # 删除缺失值所在的行
df_drop = df.dropna(axis=1) # 删除缺失值所在的列
# 填充缺失值
df_fill = df.fillna('-') # 填充缺失值，使用指定值填充
df_fill = df.ffill()    # 填充缺失值，使用前一行数据填充
df_fill = df.bfill()    # 填充缺失值，使用后一行数据填充

# 重复值处理
df.duplicated() # 查看重复值(所有列重复才会True)
df.duplicated(subset=['订单号'])   # 查看重复值(指定列重复)
# 删除重复值
df_drop_duplicates = df.drop_duplicates(subset=['订单号'], keep='first') # 删除重复值, keep='first'表示只保留第一个重复值, keep='last'表示只保留最后一个重复值, keep=False表示删除所有重复值

# 异常值处理
df[df['单价'] <= 0]
# 删除异常值
df_drop_outliers = df.drop(df[df['单价'] <= 0].index)
# 修复异常值
# 此处只举例日期异常格式修改，其它数据如何修复根据具体情况而定
df['订单日期']
df['订单日期'].str.replace('/', '-') # 通过str访问器访问到原生str的方法
```
