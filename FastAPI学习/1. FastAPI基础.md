## 项目创建

- 常规创建项目即可，没有像Maven那样麻烦的设定

- 项目启动：`uvicorn main:app --reload`  -- `--reload`代表在代码修改时自动重新加载服务器

- 交互式文档：项目端口网页后加上`/docs`  -- 可用于查看API文档和测试API

- 入门程序：
```py
    from fastapi import FastAPI

    # 创建FastAPI实例
    app = FastAPI()

    @app.get("/hello")   # 路由装饰器，指定路由为/hello，方法为GET
    async def root():
        return {"message": "Hello World666"}    # 返回JSON格式的响应（字典）
```


## 参数

### 参数注解

**对参数的形式做描述，限制，规范**

- `Path()` 为路径参数类型注解，需从`fastapi`中导包引入`Path`
- `Query()`为查询参数类型注解，需从`fastapi`中导包引入`Query`
- `Field()`为请求体参数类型注解，需从`pydantic`中导包引入`Field`，注解写在实体类中。

**常用属性**：

| 类型注解属性          | 说明          |
| --------------------- | ------------- |
| ...                   | 必填          |
| gt/ge                 | 大于/大于等于 |
| lt/le                 | 小于/小于等于 |
| description           | 描述          |
| max_length/min_length | 参数长度限制  |


### 路径参数

- 使用场景：指向唯一，特定的资源时
- 格式    ：`url/{参数}`

```py
    from fastapi import FastAPI, Path
    # ... 创建实例省略

    @app.get("/book/{id}")
    async def get_book(id: int = Path(...,
                                    gt=0,
                                    lt=101,
                                    description="图书编号在1-100之间")): # Path(): 为类型注解, ...代表必填
        return {"id": id,
                "title_num": f"当前书编号为{id}"}
```

### 查询参数

- 使用场景：对资源集合进行过滤，排序，分页等操作时
- 格式    ：`url?k1=v1&k2=v2`

> 不需要将参数写到请求路径上，函数执行时，形参会被自动解释为查询参数

```py
    from fastapi import FastAPI, Query
    # ... 创建实例省略

    @app.get("/student")                    # 默认值
    async def get_student(name: str = Query("张三", description="学生姓名", min_length=2, max_length=10),
                        age: int = 18):
        return {"name": name, "age": age}
```

### 请求体参数

- 使用场景：创建、更新资源、携带大量数据，如JSON时
- 格式    ：导入`from pydantic import BaseModel`，然后定义`实体类`

```py
    # 导包
    from fastapi import FastAPI
    from pydantic import BaseModel, Field
    # ... 创建实例省略

    # 定义实体类
    class User(BaseModel):
        name: str = Field(..., min_length=2, max_length=10, description="用户名长度需求2-10个字")
        password: str = Field(..., min_length=6, max_length=20, description="密码")
    
    # 请求函数
    @app.post("/register")
    async def register(user: User):
        return user
```

## 响应类型

**响应类型汇总**：

| 响应类型            | 用途                   | 示例                                |
| ------------------- | ---------------------- | ----------------------------------- |
| `JSONResponse`⭐     | 默认响应，返回JSON数据 | `return {"key": "value"}`           |
| `HTMLResponse`⭐     | 返回HTML内容           | `return HTMLResponse(html_content)` |
| `PlainTextResponse` | 返回纯文本             | `return PlainTextResponse("text")`  |
| `FileResponse`⭐     | 返回文件下载           | `return FileResponse(path)`         |
| `StreamingResponse` | 流式响应               | 生成器函数返回数据                  |
| `RedirectResponse`  | 重定向                 | `return RedirectResponse(url)`      |

> JSON响应类型：只要返回的是Python数据，如字典、列表等，就默认响应JSON


### 响应类型设置方式 -- 3种

**设置前需导包**：`from fastapi.responses import 响应类型`

**1. 装饰器中指定**：返回固定内容类型时（html，纯文本等）

- `@app` 中指定：

```py
    @app.get("/html", response_class=HTMLResponse)
    async def get_html():
        return "<h1>这是标题</h1>"
```

**2. 返回响应对象**：文件，图片，流式响应等动态响应

- `return` 中指定：

```py
    @app.get("/file")
    async def get_file():
        file_path = "./files/1.png"
        return FileResponse(file_path)
```

**3. 返回自定义对象**：

- 先定义对像
- 再`app@` 中指定：`response_model=类名`

```py
    # 自定义类型
    class Book(BaseModel):
        id: int
        title: str

    # 响应自定义类型            # 指定对象类型
    @app.get("/book2/{id}", response_model=Book)
    async def get_book2(id: int):
        return {
            "id": id,
            "title": f"id: {id}《Python Web开发实战》"
        }
```

## 异常处理

- 导包：`from fastapi import HTTPException`

- 格式：`raise HTTPException(status_code=.., detail="..")`

> status_code 为必填参数
> detail 为异常描述

```py
    # 导包
    from fastapi import FastAPI,HTTPException

    @app.get("/book3/{id}")
    async def get_book3(id: int):
        if id not in range(1,101):
            raise HTTPException(status_code=404, detail="图书编号不合规")

        return {"id": id,
                "title": f"id: {id}《Python Web开发实战》"}
```
