## ORM

用于在面向对象编辑语言和关系型数据库之间建立映射。它允许开发者通过**操作对象的方法**与数据库进行交互。

优势：
- 减少重复sql代码
- 自动处理数据库连接和事务
- 自动防止SQL注入攻击

ORM工具：
- SQLAlchemy ORM ⭐⭐⭐
- Django ORM
- Tortoise ORM

### 建库建表

**建库**：通过mysql创建即可，无序通过 python 或 ORM

**建表**：

1. **创建数据库引擎**：
   - 导包 `from sqlalchemy.ext.asyncio import create_async_engine` -- 导入创建异步引擎包
   - 创建异步引擎，配置SQL链接等属性

```py
    # 导包：创建数据库引擎包
    from sqlalchemy.ext.asyncio import create_async_engine

    # 数据库连接： mysql+aiomysql://数据库账号:数据库密码@数据库地址:端口/数据库名称?字符集
    ASYNC_DATABASE_URL = "mysql+aiomysql://root:123@localhost:3306/fastapi_test?charset=utf8"

    # 创建异步数据库引擎
    async_engine = create_async_engine(
        ASYNC_DATABASE_URL,
        echo=True,          # 输出SQL日志
        pool_size=10,       # 连接池大小
        max_overflow=20     # 连接池溢出大小
    )
```


2. **定义模型类**：
   - 导包 `from sqlalchemy.orm import DeclarativeBase, Mapped, mapped_column` -- 导入基类包，字段类型包，字段映射包
   - 创建基类   (继承 DeclarativeBase )
   - 创建公共字段类 (按需继承其它类) --> 可选
   - 创建表类   (继承公共字段类, 基类) --> 注意继承顺序

> 基类 可用于创建一些 公共字段，只要继承基类，就一并继承其字段
> 然而这种使用方式不够灵活，所以一般情况下共通字段是会被单独提取出来的

```py
    # 导包
    from sqlalchemy.orm import DeclarativeBase, Mapped, mapped_column   # 导入基类包，字段类型包，字段映射包
    from sqlalchemy import DateTime, func, String, Float    # 导入映射字段类型包

    # 定义模型类： 基类 + 表对应的模型类
    # 基类：
    class Base(DeclarativeBase):
        pass    # 不需要代码，仅仅是为了继承DeclarativeBase
    
    # 公共字段 --> 创建时间，更新时间
    class TimestampMixin:
        create_time: Mapped[datetime] = mapped_column(DateTime, insert_default=func.now(), default=func.now(),
                                                    comment="创建时间")
        update_time: Mapped[datetime] = mapped_column(DateTime, insert_default=func.now(), default=func.now(),
                                                    onupdate=func.now(), comment="更新时间")

    # 书籍表模型类：id，书名，作者，价格，出版社
    class Book(TimestampMixin, Base):
        # 设置表名
        __tablename__ = "book"
        # 设置字段
        id: Mapped[int] = mapped_column(primary_key=True, autoincrement=True)
        title: Mapped[str] = mapped_column(String(225), comment="书名")         # 字段类型来自 
        author: Mapped[str] = mapped_column(String(225), comment="作者")
        price: Mapped[float] = mapped_column(Float, comment="价格")
        publish: Mapped[str] = mapped_column(String(225), comment="出版社")
```


1. **创建表**：
   - 从连接池获取异步链接，开启事务，执行ORM操作
   - FastAPI应用启动时，创建表

```py
# 导包
    from contextlib import asynccontextmanager
    from fastapi import FastAPI

    # 定义建表函数
        async def create_tables():
            # 获取异步引擎，创建事务，建表
            async with async_engine.begin() as conn:
                await conn.run_sync(Base.metadata.create_all)   # 基类元数据创建表

        # 建表时机：FastAPI启动时创建
        @asynccontextmanager
        async def lifespan(app: FastAPI):
            await create_tables()
            yield

        app = FastAPI(lifespan=lifespan)
```
