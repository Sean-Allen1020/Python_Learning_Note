### 内置变量

**`__name__`：**

**说明**：意为当前模块名

如果在当前模块内部运行，`__name__`则为`main.py`
在其它模块被调用运行时，则会变成该模块原本的名字

利用该特性，可以进行单体测试：
- 当前模块使用`if __name__ == "main.py":`，可以让该if下的代码只在当前模块运行时执行，被调用时就不会执行
- IDE输入`main`，可以自动生成该if语句


**`__all__`：**

**说明**：用于设定模块导入时`*`所指的功能

`from 模块名 import *`默认的是导入模块的所有功能
而`__all__`可以指定 * 所代表的功能

`__all__ = ["功能1", "功能2", "功能5"]` ==> 没有被写进列表的功能就不会被导入

> 当以导包形式书写`from 包名 import *`时，必须在`__init__.py`文件下配置`__all__`，否则会报错


**`__dict__`：**

**说明**：python自定义类实例的一个内置变量，用于存储对象的属性

### 类与对象

**语法**：
```py
# 定义类
class 类名:
    # 类属性 -- 所有实例共享的静态变量
    属性 = 值

    # 初始化方法(构造方法)
    def __init__(self, 成员属性):
        self.属性1 = 属性1
        self.属性2 = 属性2
        ...
    
    # 成员方法
    def 方法名(self, 形参):
        ...


# 创建实例
实例名 = 类名(属性值)
```

**示例**：
```py
class Car:
    # 类属性，所有实例共享的静态变量
    wheels = 4
    tax_rate = 0.1

    # __init__初始化方法，会在实例被创建时自动调用
    def __init__(self, c_brand, c_name, c_price):   # 类似java构造体。形参实际上不需要 c_ 也可以，示例暂时这么用
        self.brand = c_brand
        self.name = c_name
        self.price = c_price

    def running(self):
        print(f"{self.brand}{self.name}正在行驶")


c1 = Car("BMW", "X5", 500000)
print(c1.__dict__)
c1.running()
```

**魔法方法**：
- `__init__`：初始化方法
- `__str__`：字符串表示的方法，类似 Java的`toString()`，设置后再打印对象就会打印字符串而非hash值了
- `__eq__`：对象比较
- `__lt__, __le__, __gt__, __ge__`：对象比较

> eq 默认对比的是两个对象的地址值
> lt，le，gt，ge 默认自定义对象不能比较，会报错
>
> 所以需要通过实现 eq和其它大小比较魔法方法，来实现比较

```py
    def __str__(self):
        return f"{self.brand} {self.name} {self.price}"

    def __eq__(self, other):
        return self.brand == other.brand and self.name == other.name and self.price == other.price

    def __lt__(self, other):
        return self.price < other.price
```
